# Uncommitted Voice Configuration Settings: Review
 
Use the **Uncommitted Voice Configuration Settings** dialog box to review all of the changes that you have made. Scroll through the dialog box to review all of the uncommitted changes, or select the arrows to expand or collapse the different sections. When you finish, select **Close**. After you close this dialog box, you can modify your changes or commit them.
  
For details about the different procedures that you can perform by using the Skype for Business Server Control Panel, see [Manage Skype for Business Server 2015](../../manage/manage.md).
