# Mcus view
 
The Mcus view stores information about the MCUs that are participating in conference sessions. This view was introduced in Microsoft Lync Server 2013.
  
|**Column**|**Data Type**|**Details**|
|:-----|:-----|:-----|
|**McuId** <br/> |int  <br/> |Unique number identifying the MCU.  <br/> |
|**McuUri** <br/> |nvarchar(450)  <br/> |URI of the MCU.  <br/> |
|**McuUriType** <br/> |nvarchar(256)  <br/> |Type of MCU URI. For more information, see the [UriTypes table](uritypes.md). <br/> |
