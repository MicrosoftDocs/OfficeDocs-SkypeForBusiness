# Toll free dialing restrictions in Indonesia

|**Details**|**Description**|
|:-----|:-----|
|Is toll-free calling available?  <br/> |Yes  <br/> |
|Dialing format(s)  <br/> | 001 803 CCC XXXX <br/>  007 803 CCC XXXX <br/> |
|Country/region ISO code  <br/> |ID  <br/> |
|Island/territory covered  <br/> |Not applicable  <br/> |
|Island/territory not covered  <br/> |Not applicable  <br/> |
|Wired/fixed line availability  <br/> |Yes  <br/> |
|Mobile network availability  <br/> |Partial. See comments.  <br/> |
|Comments, restrictions, and access issues  <br/> |Two number formats are provided for greater coverage. The number format 007 803 CCC XXXX format is accessible from the PT Telkom network, including Telkomsel and XL mobile. The number format 001 803 CCC XXXX is accessible from the Indosat network. For mobile: Matrix Cellular (prefix 0855, 0858, 0815, and 0816) only.  <br/> |
