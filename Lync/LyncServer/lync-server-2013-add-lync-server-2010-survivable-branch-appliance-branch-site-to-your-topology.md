# Add Lync Server 2010 Survivable Branch Appliance branch site to your topology

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Topic Last Modified:** 2012-10-07_

Microsoft Lync Server 2010 Survivable Branch Appliances (SBA) can be associated to a Microsoft Lync Server 2013 Front End pool as a backup Registrar. Refer to the Migration section, [Connect a Survivable Branch Appliance](connect-a-survivable-branch-appliance.md), for detailed procedures on how to connect your Microsoft Lync Server 2010 SBA to a Microsoft Lync Server 2013 Front End pool.
