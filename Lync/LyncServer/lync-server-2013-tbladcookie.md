# tblADCookie in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Topic Last Modified:** 2012-06-25_

tblADCookie contains the current Lightweight Directory Access Protocol (LDAP) Sync cookies.
