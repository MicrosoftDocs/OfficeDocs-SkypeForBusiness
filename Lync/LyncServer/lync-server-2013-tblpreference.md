# tblPreference in Lync Server 2013

</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**Topic Last Modified:** 2012-09-24_

tblPreference contains the users’ client preferences. This is generally used by clients previous to Lync 2013.
